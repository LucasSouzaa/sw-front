## Instalando projeto localmente

Clonar o repositorio
```git clone https://github.com/LucasSouzaa/sw-front.git```

Acessar pasta do projeto
```cd sw-front```

Instalar as dependencioas
```npm install```

Rodar o projeto
```npm start```
